# 📊 Customer-Insights-Statistical-Investigation

A statistical analysis and exploratory data analysis project using Python to understand customer behavior, spending patterns, engagement trends, and demographic insights through hypothesis testing.

---

## Project Overview

This project focuses on performing statistical analysis and exploratory data analysis (EDA) on a customer dataset to uncover meaningful business insights. The objective is to analyze customer demographics, spending behavior, engagement patterns, and validate business assumptions using statistical hypothesis testing.

The project demonstrates practical implementation of:

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Descriptive Statistics
- Hypothesis Testing
- Data Visualization
- Business Insight Generation

---

## Business Problem Statement

A retail company wants to better understand customer behavior and spending patterns using data analytics and statistical methods. This project helps identify trends, demographic influences, and customer engagement opportunities to support data-driven business decisions.

---

## Dataset Features

The dataset contains customer demographic and behavioral information including:

- CustomerID
- State
- Education
- Gender
- Age
- Marital Status
- Number of Pets
- Monthly Spend
- Days Since Last Interaction
